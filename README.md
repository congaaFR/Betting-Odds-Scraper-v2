# 🎯 Betting Odds Scraper

Scraper for coteur.com odds with colored display and copy-paste ready tables.

## 🚀 Quick Start

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## 🎨 Features

- ⭐ Auto-selected favorite competitions
- 🎰 Fixed bookmaker order: Winamax | Unibet | Betclic | PMU | Betsson
- 🎨 Colored view with brand colors
- 📋 Copy-paste mode for Google Sheets
- ✅ Empty cells for missing odds

## 📊 Display Modes

- **Colored View** - See bookmakers with their brand colors
- **Copy-Paste View** - Clean table ready to paste

Choose in the sidebar!
